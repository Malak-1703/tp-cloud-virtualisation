# TP1 : Hyperviseur ESXi vs Proxmox
## Objectif
Explorer et sécuriser les hyperviseurs Type 1.
## Réalisations
- Création de VMs et configuration réseau (vSwitch / Bridges).
- Mise en place du RBAC (Utilisateurs limités).
- Prise de snapshots et compréhension de la différence avec les backups.
## Sécurité
- Segmentation Management / VM.
- Désactivation SSH si non nécessaire.
