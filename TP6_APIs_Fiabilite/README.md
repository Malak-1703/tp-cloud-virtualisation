# TP6 : APIs, Communication et Fiabilité
## Objectif
Concevoir des APIs robustes et résilientes face aux pannes réseau.
## Réalisations
- Spécification de contrats d'API (HTTP, JSON, codes d'erreur).
- Implémentation de patterns de fiabilité : Timeouts, Retries, Backoff exponentiel avec Jitter.
- Analyse de l'idempotence pour éviter les doublons (ex: double paiement).
