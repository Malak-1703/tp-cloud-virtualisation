# TP5 : Concepts des Systèmes Distribués
## Objectif
Comprendre les fondements des architectures distribuées (CAP, Zero Trust, pannes partielles).
## Réalisations
- Modélisation d'une architecture micro-services (Système de Gestion Documentaire).
- Analyse des défis distribués : latence, cohérence, tolérance aux pannes.
- Cartographie des surfaces d'attaque et application du modèle Zero Trust.
